# 🚀 Antigravity Rules: AI Assistant Guidelines

- **Modern Swift:** iOS 17.0 이상을 타겟팅하며, 테스트 코드는 XCTest가 아닌 **`Swift Testing`** 매크로(`@Test`)를 기본으로 작성한다.
- **Ephemeral Architecture:** 데이터는 명시적인 요구가 없는 한 영구 저장(SwiftData/CoreData)을 피하고, 메모리 위에서만 처리된 후 안전하게 소멸(Free)되도록 설계한다.
- TODO.md 파일이 있다면, Phase 마다 feature branch를 만들어서 작업을 시작해야하고 TODO 한 개를 체크 완료할 때마다 commit message를 함께 추천해준다.
